Instructions

Overview of Peabody Picture Vocabulary Test 4 (PPVT-IV)

The Peabody Picture Vocabulary Test-4 (PPVT–4) scale is a norm-referenced, wide-range instrument for measuring the receptive (hearing) vocabulary of children and adults.

Enlarged and colorized, this PPVT edition is available in two parallel forms (Form A and Form B) that are administered individually. Each form contains training items and 228 test items, each consisting of four full-color pictures as response options on a page. For each item, the examiner says a word, and the examinee responds by selecting the picture that best illustrates that word’s meaning.

How to Start

Before each exam, ensure the examinee knows how the test will work by administering the training items in the easel.
Start with the set that has the age of the examinee in the upper left corner. This is the basal set.

Basal Set Rule: The examinee may only make 1 or 0 errors on the basal set. If necessary, the administrator should move back sets until a basal set can be established or until Set 1 is completed.

Conducting the test

Ceiling Set Rule: Stop the test after a student fails a set that is not a basal set. Examinees fail a non-basal set by missing 8 or more questions from that set.
Complete Set Rule: Every set that is started must be finished (all 12 items in the set must be given in order.)

Scoring

Calculate the total number of errors made including errors made in the basal set and ceiling set (the last set given), but do not include errors made in sets before the basal set was established.
Calculate the raw score by subtracting the total number of errors made from the Ceiling item. The Ceiling item is the number of the last word in the examinee's ceiling set. For example, if the examinee's ceiling set was Set 6, the ceiling item is 72.
